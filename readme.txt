Delete post using ajax without page load.

== Description ==

This plugin is used for remove post using ajax in WordPress admin dashboard without page refresh.


== Installation ==

1. Download the plugin and unzip it.
2. Upload the folder remove-post-using-ajax-admin/ to your wp-content/plugins folder.
3. Active the plugin from your WordPress admin panel/
4. Installation Done.
